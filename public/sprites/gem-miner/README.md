# Gem Miner Starter Sprite Pack

Replace any file in this folder with your own art while keeping the same file names.

Expected canvas size:
- Tile sprites: 24x24
- Robot sprites: 24x24
- Depot sprites: 56x40
- UI sprites (upgrade icons): around 34x22
- FX sprites: 8x8 to 24x24

If dimensions differ, the game will scale to fit tile size automatically.
